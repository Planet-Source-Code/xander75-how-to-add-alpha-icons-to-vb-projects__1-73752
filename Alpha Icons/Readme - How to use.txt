Add Alpha Icons to any Project from small Form 16x16 icons to the larger compiled desktop icons.

How to use:

1. Open the Resource folder...
2. Right click on the "AlphaIcon.rc" file and click Edit, open this with Notepad.
3. Add as many different icons you want to add, then save the file.
4. Run the MAKERES.BAT file, this will create a Resource file called AlphaIcon.res, 
   when performing this it might be best to delete the existing AlphaIcon.res first!
5. Add the resource file into your VB Project.
6. Follow the easy to use code in the FormLoad on frmMain.
7. Complie your project, there you go... you now have Alpha Icons in your VB Projects!
